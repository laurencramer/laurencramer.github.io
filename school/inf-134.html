project in user interface software

Winter 2019 - 6 weeks

about

The project was to design a new application to solve a problem that our group thought was missing from the market. We created Cheqin, a mood-tracking mobile application that was designed to make mood-tracking easier and provide easily understandle data to increase the benefits of mood-tracking. We were required to conduct usability tests, create a competitive analysis, create prototypes and wireframes, and produce a final design.
